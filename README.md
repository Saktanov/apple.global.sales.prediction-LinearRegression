# 📊 Apple Sales Prediction

## Описание проекта

Данный проект представляет собой систему предсказания выручки (revenue) на основе данных о продажах продукции Apple с использованием методов машинного обучения.

Модель анализирует числовые характеристики продаж и прогнозирует ожидаемую выручку. Проект включает обучение модели, предсказание через CLI и веб-интерфейс на Streamlit для визуализации результатов.

---

## Цель проекта

Цель проекта — разработать простую и наглядную систему для прогнозирования выручки на основе ключевых факторов, таких как:

* цена продукта
* скидка
* количество проданных единиц

## Установка и запуск

### 1. Установка зависимостей

```id="g2m8ya"
pip install pandas scikit-learn matplotlib streamlit joblib
```

---

### 2. Обучение модели

```id="j8l2dk"
python train.py
```

После выполнения:

* обучится модель
* создастся `model.pkl`
* сохранятся графики в папке `static/`
* создастся файл `metrics.txt`

---

### 3. CLI предсказание

```id="k9x2pw"
python predict.py
```

Функционал:

* ввод данных на русском языке
* защита от некорректного ввода
* вывод предсказанной выручки

---

### 4. Запуск веб-приложения

```id="m7z3qa"
streamlit run app.py
```

После запуска открой в браузере:

```
http://localhost:8501
```

## Использование

Проект поддерживает 3 основных сценария:

---

## Как работает модель

В проекте используется:

* **LinearRegression** — для предсказания выручки
* разбиение данных на train/test
* оценка качества модели

---

## Метрики модели

Используются:

* **R² (коэффициент детерминации)** — показывает качество модели
* **MAE (Mean Absolute Error)** — средняя ошибка

Метрики сохраняются в файл `metrics.txt` и отображаются в веб-приложении.
